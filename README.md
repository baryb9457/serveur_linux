# Serveur Linux — Accès SSH à distance

Mise en place d'un environnement **Linux Server** accessible à distance via SSH.

![Capture du serveur Linux](https://github.com/user-attachments/assets/fea5206d-5690-438e-aed3-c48713089555)

## Description

Ce projet documente la configuration d'un serveur Linux permettant une connexion sécurisée à distance via le protocole SSH (Secure Shell).

## Technologies utilisées

- **Système** : Linux (Debian/Ubuntu Server)
- **Protocole** : SSH (port 22 / port personnalisé)
- **Sécurité** : Authentification par clé, pare-feu UFW

## Ce qui a été mis en place

- Installation et configuration du service SSH (`openssh-server`)
- Restriction d'accès (utilisateurs autorisés, désactivation du login root)
- Configuration du pare-feu UFW
- Connexion distante et administration à distance via terminal

## Compétences démontrées

Administration système Linux, réseaux, sécurité informatique, accès distant.
